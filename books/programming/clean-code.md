# 📖 Clean Code – Robert C. Martin

## ⭐ Rating
8/10

## 🧠 Key Takeaways
- Code should be readable, not just functional
- Functions should do one thing
- Naming is extremely important

## 🔧 Practical Applications
- Refactor messy scripts into clean modules
- Use meaningful variable/function names
- Reduce complexity in projects

## 💻 Projects I Can Build From This
- Refactor an old GitHub project
- Create a clean CLI tool
- Build a structured Python app

## ⚠️ Weak Points / Criticism
- Some ideas are opinion-based
