## Weekly Schedule

| Day | Focus | Training |
|-----|-------|----------|
| MON | Back & Biceps | 2-mi easy run (warm-up) · Deadlift 4×6 · Barbell Row 4×8 · Wide-Grip Pull-Ups 4×6 · DB Curl 4×8 · BB Curl 4×10 · Hammer Curl 4×10 · Cable Crunch 3 sets |
| TUE | Run + Mobility | 4-mi run (mod) · 20 min mobility & stretching — no lifting |
| WED | Chest & Triceps | 2-mi easy run (warm-up) · Bench 4×6 · Incline DB 3×8 · Weighted Dips 3×8 · Cable Pulldown 3×10 · Triceps Ext 3×10 · Ab work |
| THU | Run + Mobility | 4-mi run (mod) · 20 min mobility & stretching — no lifting |
| FRI | Legs | 2-mi easy run (warm-up) · Squat 4×6 · Front Squat 4×5 · Hip Thrust 4×8 · Leg Curl 3×10 · Leg Ext 3×10 · Calf Raise 3×10 · Cable Crunch 3 sets |
| SAT | Shoulders | 2-mi easy run (warm-up) · DB Shoulder Press 4×10 · Arnold Press 4×10 · Lateral Raise 4×10 · Front Raise 2×12 · Ab Cable Pulldown 4×10 |
| SUN | Long Run + Rest | 4-mi run (easy) · Light mobility · Meal prep for the week |

**Daily:** Push-ups 5-6 × 15-18 (never to failure) · Abs rotated: ab wheel / cable crunch / hanging leg raise

---

## Intensity & Progression

| Goal | Load | Reps |
|------|------|------|
| Strength | Working weight | 4-6 |
| Hypertrophy | ~80% of above | 8-12 |

**Progression rule:** Can't reach top of range → hold weight. Hits top easily → add 5 lbs next session.
